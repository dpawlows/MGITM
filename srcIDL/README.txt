Compile various macros by running idlr on startup, or using

.r gitm_procedures

read_thermosphere_file.pro reads an M-GITM file.

See example usage on line 20 of thermo_batch.pro, which can be used to quickly
plot output.

thermo_batch.pro can be run in the data directory and it is prompt driven.
Don't have to send in any arguments. 

thermo_batch_v3.pro and thermo_plotbatch_v3.pro are from Steve Bougher.
